
README Projet Groupe 4
Bienvenue dans le projet du Groupe 4, composé de SILVA SAMUEL, ZIDI YANIS, BAIROUKI YSSAM, TOUGRI HICHAM, WOHL NATHAN pour l'année Pré-ing 2 2023.

Ce document README a pour but de vous guider à travers l'installation, la configuration et l'utilisation de notre application. Veuillez suivre les instructions ci-dessous pour une expérience utilisateur optimale.

Prérequis

Notre projet est conçu pour fonctionner avec le serveur WAMP, nous recommandons donc d'installer cette plateforme si ce n'est pas déjà fait. Vous pouvez télécharger WAMP Server à partir du site officiel de WAMP.

Assurez-vous que votre ordinateur respecte les conditions suivantes pour un affichage optimal :

Résolution d'écran : 1920x1080
Echelle de rendu d'affichage de l'écran à 100%
Un navigateur moderne est recommandé pour une expérience optimale, par exemple Google Chrome.

Installation
Après avoir installé WAMP, procédez comme suit pour installer le projet :
Démarrez WAMP Server.
Copiez le dossier "Projet" dans le répertoire www de WAMP (généralement situé dans le répertoire C:\wamp64\www).
Ouvrez votre navigateur et accédez à http://localhost/Projet/accueil.php.
Résolution des problèmes
Si des problèmes d'affichage surviennent lors de l'utilisation du site, les causes peuvent être diverses. Voici quelques étapes de dépannage que vous pouvez essayer :

Résolution d'écran : Assurez-vous que votre résolution d'écran est réglée sur 1920x1080. Vous pouvez changer la résolution d'écran dans les paramètres d'affichage de votre système d'exploitation.
Echelle de rendu d'affichage de l'écran : Assurez-vous que votre échelle de rendu est réglée à 100%. Vous pouvez changer l'échelle de rendu dans les paramètres d'affichage de votre système d'exploitation.
Cache du navigateur : Si le problème persiste, vous pouvez essayer de vider le cache de votre navigateur. Sur Google Chrome, vous pouvez le faire en ouvrant la console (F12), en allant dans l'onglet "Network" (Réseau), en cochant la case "Disable cache" (Désactiver le cache) et en actualisant la page (F5).
